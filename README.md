# Calculator

## Description
A fully functional calculator built using HTML, CSS, and JavaScript.  
It supports basic arithmetic operations and updates the display dynamically as the user interacts with the buttons.

## Features
- Addition, subtraction, multiplication, division
- Live display updates
- Clear/reset functionality
- Keyboard-friendly layout
- Responsive and clean UI

## Technologies Used
- HTML5
- CSS3 (Grid layout for button arrangement)
- JavaScript (event handling + calculation logic)
- Git & GitHub for version control
